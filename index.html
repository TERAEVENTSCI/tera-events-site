Synchronisation non configurée

Configurez JSONBin.io pour synchroniser les données entre les utilisateurs en temps réel.

Guide de configuration

1

Créer un compte JSONBin.io (gratuit)

Allez sur jsonbin.io → Sign Up → connectez-vous à votre tableau de bord

2

Récupérer votre X-Master-Key

Menu → API Keys → copiez votre X-Master-Key (commence par $2a$...)

3

Créer le Bin automatiquement

Collez votre X-Master-Key ci-dessous puis cliquez "Créer le Bin" — le BIN ID sera rempli automatiquement. Partagez ensuite ce BIN ID avec Eric.

Configuration JSONBin.io

Créez le Bin une seule fois (depuis le compte Super Admin) puis partagez le BIN ID avec Eric.
